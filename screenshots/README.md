## 檔案格式

- 支援格式：`.png`、`.jpg`、`.jpeg`
- 建議使用 `.png` 格式以獲得最佳畫質

## 上傳方式

1. 將截圖檔案放入此資料夾
2. 確認檔案名稱完全符合上述規範（包括編號和連字號）
3. 提交並推送到 GitHub

## 驗證

GitHub Actions 會自動檢查此資料夾中的截圖是否完整。所有 8 張截圖都存在時，進階任務驗證才會通過。
